# App assesment with fastAPI

## System requirements
* Using Python 3.9.4
* Using MongoDB 6.0

## Get started

### Install dependencies
`pip install -r requirements.txt`

### Run Mongodb localhost:27017
`mongod`

### Run the app
`uvicorn main:app`

## Swagger docs
You should go to the path in the browser: http://localhost:8000/docs
